🏦 Loan Approval Data Analysis
📌 Project Overview

This project focuses on analyzing loan approval data to understand factors affecting loan approval decisions. The goal is to identify patterns related to applicant income, credit history, loan amount, and other features.

📂 Dataset
Loan approval dataset
Includes applicant income, loan amount, credit history, gender, education, etc.

🧹 Data Cleaning
Handled missing values
Converted categorical variables
Removed inconsistencies
Data type corrections

📊 Analysis Performed
Loan approval vs income
Credit history impact
Loan amount distribution
Gender & education analysis
Approval rate comparison

📈 Key Insights
Credit history strongly impacts loan approval
Higher income increases approval chances
Certain applicant categories have higher approval rates
Loan amount affects approval probability

🛠️ Tools Used
Python 🐍
Pandas
NumPy
Matplotlib
Seaborn
Google Colab

🚀 Conclusion
This project helps understand the key factors influencing loan approval decisions and demonstrates data analysis skills using real-world financial data.
